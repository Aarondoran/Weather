# Contributing to Static Website Template

Thank you for considering contributing to this project! Your contributions help improve the template for everyone.

---

## How to Contribute

I accept contributions through **forks and pull requests**:

1. Fork this repository.  
2. Create a new branch for your changes.  
3. Make your changes (add pages, update styles, scripts, images, or documentation).  
4. Ensure your changes **function correctly**.  
5. Submit a pull request describing your changes clearly.

---

## What You Can Contribute

- Fixes to existing files.  
- New pages or templates.  
- Styles or scripts.  
- Images or other assets.  
- Documentation improvements.  

All contributions that adhere to these guidelines are welcome.
